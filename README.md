# 🌊 Memory Tide

A conceptual web project exploring a spatial approach to archiving and interacting with personal memories.

👉 **Live Demo:** https://xy-memory-tide-soea.vercel.app/trace  
👉 **GitHub:** https://github.com/ayouuuu2-cpu  

---

## The Concept

Traditional note-taking tools organize memories in linear timelines or fixed folders. This often limits how we naturally recall and connect experiences.

Memory Tide explores an alternative approach—treating memories as spatial points within a navigable canvas. Users can place text, images, and locations as anchors and revisit them through spatial exploration instead of linear scrolling.

---

## Core Features

- **Spatial Mapping:** Pin text and image memories to spatial nodes  
- **Canvas Exploration:** Navigate through memories in a visual environment  
- **Contextual Linking:** View entries based on spatial relationships  

---

## Tech Stack

- **Framework:** Next.js / React  
- **Deployment:** Vercel  

---

## Future Roadmap

This is an early-stage prototype focused on validating the core interaction idea. Future work will focus on:

- improving canvas interaction and transitions  
- refining spatial relationships between nodes  
- enhancing overall structure and visual clarity
