## The Concept

Traditional note-taking tools organize memories in linear timelines or fixed folders. This often limits how we naturally recall and connect experiences.

Memory Tide explores a different approach—treating memories as spatial points within a navigable canvas. Users can place text, images, and locations as anchors and revisit them through spatial exploration instead of linear scrolling.

---

## How to use

1. Open the live demo  
2. Click on the canvas to create a memory node  
3. Add text or image content  
4. Explore memories spatially through navigation  

---

## Core Features

- Spatial memory placement (text + images)  
- Interactive canvas-based exploration  
- Context-based navigation instead of list view
